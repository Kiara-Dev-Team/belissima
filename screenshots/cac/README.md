# CAC (Customer Acquisition Cost) Screenshots

Place the following screenshots in this directory:
- `chartjs-cac.png` - Chart.js bar chart (monthly CAC)
- `plotly-correlation.png` - Plotly.js bar + scatter (CAC vs conversion)
- `d3-cac-trend.png` - D3.js bar + trend line with annotations

## Specifications
- Resolution: 1200x600px minimum
- Format: PNG with transparency
- Color palette: Dual-Purpose (Red/Green)
- Show: Monthly data with moving average trend
