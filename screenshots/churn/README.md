# Churn Rate Screenshots

Place the following screenshots in this directory:
- `chartjs-churn.png` - Chart.js line chart (trend)
- `plotly-churn-combo.png` - Plotly.js combo chart (context)
- `d3-waterfall.png` - D3.js waterfall chart (cohort flow)

## Specifications
- Resolution: 1200x600px minimum
- Format: PNG with transparency
- Color palette: Neutral Professional (with red accent)
- Show: Monthly churn rate with benchmark lines
